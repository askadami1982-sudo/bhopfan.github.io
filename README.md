# Tuxrak Bhop Server Maker

Fan yapımı GitHub Pages demosu.

## Özellikler
- Create Server butonu
- 4 haneli server kodu üretme
- `.bhop` custom map yükleme
- 3D parkur önizleme
- Örnek `ataparkour.bhop`

## GitHub Pages'e koyma
1. Bu klasördeki dosyaları GitHub reposuna yükle.
2. Settings > Pages > Deploy from branch seç.
3. Branch: main, folder: /root seç.
4. Site linkin açılınca `index.html` çalışır.

## Not
Bu proje Bhop Pro resmi serverlarına bağlanmaz. Fan yapımı web panelidir.
